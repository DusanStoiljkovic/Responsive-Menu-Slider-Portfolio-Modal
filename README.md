# Responsive Menu, Slider & Portfolio Modal

## Overview
This project is a responsive front-end interface that includes a mobile-friendly navigation menu, an interactive image slider, a portfolio showcase, and modal popups for detailed views. It is built with HTML, CSS, and vanilla JavaScript, focusing on responsive design and user experience.

## Features
- Responsive navigation menu optimized for desktop and mobile
- Smooth and interactive image slider/carousel
- Portfolio section with clickable items
- Modal popups displaying project details
- Clean and semantic HTML structure
- CSS animations and transitions for enhanced UI feedback

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (ES6)
